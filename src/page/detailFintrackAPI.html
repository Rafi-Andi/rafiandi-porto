Deskripsi proyek

Smart Financial Tracker API adalah backend aplikasi pelacakan keuangan cerdas yang menyediakan berbagai fitur untuk mencatat transaksi harian, menampilkan ringkasan keuangan, hingga memberikan prediksi saldo masa depan dengan bantuan AI. API ini dirancang untuk mendukung aplikasi front-end dan memungkinkan pengguna mengelola keuangan mereka secara efisien melalui layanan seperti registrasi, login, pencatatan transaksi, chatbot AI, dan prediksi saldo berbasis machine learning.
