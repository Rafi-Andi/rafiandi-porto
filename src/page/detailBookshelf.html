Deskripsi proyek

API Bookshelf adalah sebuah layanan RESTful untuk mengelola koleksi buku digital. API ini memungkinkan pengguna melakukan operasi CRUD (Create, Read, Update, Delete) terhadap data buku, dengan fitur tambahan seperti filter dan pencarian. Dengan API ini, pengguna dapat: Menambahkan buku baru ke koleksi dengan detail lengkap (judul, penulis, tahun terbit, dll.) Melihat daftar semua buku dalam koleksi dengan informasi ringkas Melihat detail lengkap dari buku tertentu berdasarkan ID Memperbarui informasi buku yang sudah ada Menghapus buku dari koleksi Mencari dan memfilter buku berdasarkan nama, status membaca, dan status penyelesaian API ini dirancang dengan pendekatan RESTful yang sederhana dan respons JSON yang terstruktur, membuatnya mudah untuk diintegrasikan dengan berbagai aplikasi front-end. Setiap buku memiliki properti seperti status "sedang dibaca" dan "selesai dibaca", sehingga cocok untuk aplikasi yang membantu pengguna melacak progres membaca mereka.
